X-UK-Gov Public Repository Leaderboard

Updated every 24 hours. Scrapes GitHub's official list of governments then fetches all their repos via the GitHub API. Pull requests welcome. The raw data is available as repos.json for your own analysis — prefer linking to it rather than downloading a copy, so your tools always use the latest data. A consolidated CycloneDX SBOM (gzipped) and individual SPDX SBOMs are collected hourly via the GitHub dependency graph API. You can also explore the data with Datasette Lite for SQL-powered querying directly in your browser.
